# Employee Headcount Trend Analysis

## Project Overview
This project analyzes workforce data to calculate monthly end-of-month (EOM) employee headcount trends. The goal is to provide HR and management with insights into staffing levels, attrition, and workforce growth over time.

## Key Features
- Calculated monthly EOM employee headcount using SQL
- Built logic to determine active employees based on start and end dates, including ongoing employment
- Accounted for attrition and re-hiring in headcount calculations
- Visualized trends to identify periods of growth, contraction, or stability

## Tools & Technologies
- SQL: Data extraction, aggregation, and headcount calculation
- Python / Pandas (optional): For additional trend analysis and visualization
- Matplotlib / Seaborn: Charts to show monthly headcount trends
- Excel (optional): For validation and reporting

## Sample Insights
- Headcount increased steadily during Q2 2024, with peak hiring in May
- Attrition spikes were observed in March and September
- Active employee distribution highlights departments with the highest retention

## Future Improvements
- Integrate automated dashboards using Power BI or Tableau for real-time monitoring
- Add departmental breakdowns to identify specific teams with high turnover
- Predictive modeling to forecast future headcount and attrition trends
